# Sree Enterprises - Textile E-Commerce Platform

🛍️ A modern, professional, and feature-rich e-commerce website for textile and fashion products.

## 📋 Project Overview

Sree Enterprises is a full-stack e-commerce application designed specifically for Indian textile and fashion businesses. It provides a seamless shopping experience for customers and powerful management tools for administrators.

### Business Type
- **Products**: Men's Clothing, Women's Clothing, Traditional Wear, Casual Wear, Sarees, Kurtas, Shirts, T-Shirts, Dresses, Fashion Accessories
- **Target Market**: Indian and International customers
- **Objective**: Increase online sales, build brand trust, premium shopping experience

## 🚀 Technology Stack

### Frontend
- React 18+ with TypeScript
- Tailwind CSS for styling
- Redux Toolkit for state management
- Framer Motion for animations

### Backend
- Node.js with Express.js
- MongoDB with Mongoose ODM
- JWT + OTP Authentication
- Razorpay Payment Integration

### Hosting
- **Frontend**: Vercel
- **Backend**: Render or AWS EC2
- **Database**: MongoDB Atlas

## 🚀 Quick Start

### Prerequisites
- Node.js v16+
- MongoDB
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ab5036166-jpg/Sree-Enterprise-.git
   cd Sree-Enterprise-
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   cp .env.example .env
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd ../frontend
   npm install
   cp .env.example .env
   npm start
   ```
