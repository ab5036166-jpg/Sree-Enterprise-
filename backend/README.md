# Backend - Sree Enterprises E-Commerce Platform\n\nExpress.js + MongoDB backend server for textile e-commerce\n\n## 🚀 Quick Start\n\n### Prerequisites\n- Node.js v16+\n- MongoDB (local or MongoDB Atlas)\n- npm or yarn\n\n### Installation\n\n1. Install dependencies\n   ```bash\n   npm install\n   ```\n\n2. Configure environment variables\n   ```bash\n   cp .env.example .env\n   ```\n\n3. Start development server\n   ```bash\n   npm run dev\n   ```\n\nServer will run at `http://localhost:5000`\n\n## 📁 Project Structure\n\n```\nsrc/\n├── config/          # Configuration files\n├── controllers/     # Route controllers\n├── middleware/      # Custom middleware\n├── models/          # Mongoose models\n├── routes/          # API routes\n├── services/        # Business logic\n├── utils/           # Utility functions\n└─�� server.ts        # Entry point\n```\n\n## 🔧 Available Scripts\n\n- `npm run dev` - Start development server with hot reload\n- `npm run build` - Build TypeScript to JavaScript\n- `npm start` - Run production build\n- `npm run lint` - Run ESLint\n- `npm run test` - Run tests\n- `npm run seed` - Seed database with sample data\n\n## 📝 Environment Variables\n\nSee `.env.example` for all available configuration options.\n\nKey variables:\n- `MONGODB_URI` - Local MongoDB connection\n- `JWT_SECRET` - JWT signing secret\n- `RAZORPAY_KEY_ID` - Razorpay API key\n- `RAZORPAY_KEY_SECRET` - Razorpay API secret\n\n## 📡 API Endpoints\n\nAll endpoints are prefixed with `/api/`\n\n### Authentication\n- `POST /auth/register` - Register new user\n- `POST /auth/login` - Login user\n- `POST /auth/send-otp` - Send OTP\n- `POST /auth/verify-otp` - Verify OTP\n\n### Products\n- `GET /products` - Get all products\n- `GET /products/:id` - Get single product\n- `GET /products/search` - Search products\n\n### Orders\n- `POST /orders` - Create order\n- `GET /orders` - Get user orders\n- `GET /orders/:id` - Get order details\n\n### Admin\n- `POST /admin/products` - Create product (admin only)\n- `PUT /admin/products/:id` - Update product (admin only)\n- `DELETE /admin/products/:id` - Delete product (admin only)\n\n## 🔐 Security\n\n- CORS protection enabled\n- Helmet.js for HTTP headers\n- JWT authentication\n- Password hashing with bcrypt\n- Rate limiting on sensitive endpoints\n- Input validation\n\n## 📦 Technologies\n\n- **Framework**: Express.js\n- **Database**: MongoDB with Mongoose\n- **Authentication**: JWT + OTP\n- **Payments**: Razorpay\n- **File Upload**: Multer\n- **Logging**: Winston\n- **Validation**: Express Validator\n\n## 🐛 Debugging\n\nSet `LOG_LEVEL=debug` in `.env` for verbose logging.\n\n## 📚 More Documentation\n\nSee the main [README.md](../README.md) and [docs/](../docs/) folder for detailed documentation.\n"
